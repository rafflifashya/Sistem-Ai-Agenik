# Sistem-Ai-Agenik
Proyek ini merupakan tugas mata kuliah Pemrograman Berorientasi Objek yang berfokus pada pembuatan Sistem AI Agenik sederhana menggunakan bahasa Python. Di dalamnya terdapat kode program berbasis OOP, diagram UML Class, laporan akhir dalam format PDF, serta dokumentasi prompt AI yang digunakan untuk pembuatan sistem.
